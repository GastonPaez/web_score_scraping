Pagina Web Resultados por Scrapy

Esta pagina web se desarrollo con python utilizando la libreria Scrapy para scrapear los datos de las tablas de posiciones del futbol argentino. Utiliza Zyte Scrapy Cloud Deploy para alojar el spider que se envia en Json. El frontend basico se realizo con materialize.
